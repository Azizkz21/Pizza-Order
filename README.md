# Pizza Order (React Pizza)

A small but complete pizza shop UI built with React + TypeScript.
Pick dough and size, sort the menu, add items to the cart, and see totals instantly.

## Features
- Pizza catalog with dough/size variants
- Category tabs and sorting (popular, price, alphabet)
- Cart with quantity controls, remove, clear, totals
- Responsive layout (mobile > desktop)

## Tech Stack
- React
- TypeScript
- Vite
- React Router
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

## Scripts
```bash
npm run dev
npm run build
npm run preview
```

## Notes
Cart state is handled via Context + useReducer.
